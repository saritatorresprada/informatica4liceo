Ejemplos de ecuaciones en HTML

Para escribir ecuaciones en una pagina web en HTML se puede usar MathJax. La fuerza electrica se puede calcular como $$\vec{F}=q\vec{E}$$

La fuerza electrica entre dos cargas es: $$\vec{F_{12}}=\frac{1}{4\pi\epsilon_0}$$

La circuitazione del campo elettrico è:

$$\Gamma(\vec{E})=\sum_{i=1}^n\vec{\Delta 1}_i\cdot\vec{E}_1$$

La solucion de una ecuacion cuadratica de la forma \(ax^2+bx+c=0\)

es la siguiente: $$x_{1,2}=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$$
